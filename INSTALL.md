## Installation
### Using VS Code UI
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Click ... menu and select Install from VSIX...
4. Select the VSIX file

### Using Command Line
code --install-extension "dist\vsix\cef-canvas-extension-0.1.0.vsix"

## Verification
After install, open Command Palette and type 'CEF Canvas'
You should see:
- CEF Canvas: Open Canvas
- CEF Execution: Open Execution View

## Dependencies Included
- TinyMCE Editor v8.3.2
- Marked, SQL.js, KaTeX, Mermaid
- All required npm packages
